# Trigger tests for pre-grill-me

## Should trigger

- "Grill me on this architecture proposal."
- "I want you to stress-test this migration plan."
- "Challenge every assumption in this design doc."
- "Interrogate me on this experiment plan before I start coding."
- "I'm not sure if this approach is solid — push back on it hard."

## Should NOT trigger

- "What do you think of this plan?" — soliciting an opinion, not a structured interrogation.
- "Help me write a project plan." — drafting, not stress-testing an existing one.
- "Review this code." — code review, off-domain for this skill.
